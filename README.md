# Aegis---USB-Remote-Executor
Project Objectives

Design a low-cost USB Human Interface Device (HID).
Detect infrared (IR) signals using an onboard sensing circuit.
Convert IR events into USB HID actions.
Provide plug-and-play functionality without requiring additional drivers.
Demonstrate reliable communication between embedded hardware and a host computer.
Key Features
USB HID-compliant device.
IR signal detection and processing.
STM32F042-based embedded controller.
Driverless operation on modern operating systems.
Compact and low-power hardware design.
Customisable HID actions through firmware.


Conclusion
The USB HID IR Trigger Device demonstrates the practical integration of embedded systems, infrared sensing, and USB communication protocols. By leveraging the STM32F042 microcontroller and the USB HID standard, the device successfully converts IR-triggered events into computer input actions without requiring custom drivers.
The project highlights the design and implementation of a compact, cost-effective, and reliable hardware solution capable of supporting automation, accessibility, and remote-control applications. Through successful USB enumeration, IR detection, and HID report generation, the project validates the feasibility of using embedded USB devices for real-world human-computer interaction scenarios.
